# Minesweeper

A simple Minesweeper implementation using Tkinter and vibec coding with modification.

Files
- [minesweeper.py](minesweeper.py) — main game implementation (see classes [`Minesweeper`](minesweeper.py) and [`Cell`](minesweeper.py)).
- [minesweeper.pyproj](minesweeper.pyproj) — project file for Visual Studio / Python Tools.

Requirements
- Python 3.6+
- tkinter (usually included; on Debian/Ubuntu: `sudo apt-get install python3-tk`)

Run
1. Open a terminal in the repository root (where [minesweeper.py](minesweeper.py) is located).
2. Run: python minesweeper.py
